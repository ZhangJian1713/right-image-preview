# 键盘快捷键

[English](./keyboard.md) · **中文**

---

预览打开后，焦点自动锁定在遮罩层上，以下快捷键立即生效。

> 当缩放比例输入框处于编辑状态时，方向键和空格不触发全局快捷键，由输入框自行处理。

## 缩放

| 按键 | 功能 |
|------|------|
| `+` / `=` / `↑` | 放大（跳到下一个较大档位） |
| `-` / `↓` | 缩小（跳到下一个较小档位） |
| `0` | 切换到适应（Fit）模式 |
| `1` | 切换到原始比例（100%） |
| `Space` | 在适应 ↔ 100% 之间切换（与双击效果相同） |

> 缩放比例输入框可输入正整数，提交时会**限制在当前 `stops` 的最大值**（默认最高 200%），按 **Enter** 确认，**Esc** 取消。

## 图片导航

| 按键 | 功能 |
|------|------|
| `←` | 上一张（组内；无 `groupedImages` 时全局） |
| `→` | 下一张（组内；无 `groupedImages` 时全局） |
| `PageUp` | 跳到上一组第一张（需非空 `groupedImages`） |
| `PageDown` | 跳到下一组第一张（需非空 `groupedImages`） |

## 旋转

| 按键 | 功能 |
|------|------|
| `Ctrl / ⌘` + `←` | 逆时针旋转 90° |
| `Ctrl / ⌘` + `→` | 顺时针旋转 90° |

## 关闭

| 按键 | 功能 |
|------|------|
| `Esc` | 关闭预览 |

---

> 任意按键操作均会重置控件自动渐隐计时器，立即将所有控件恢复为完全可见状态。
